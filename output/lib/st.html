Library

To get started, just file in thrift.st with Squeak, run thrift -st on the tutorial .thrift files (and file in the resulting code), and then:

calc := CalculatorClient binaryOnHost: ‘localhost’ port: ‘9090’ calc addNum1: 10 num2: 15

Tested in Squeak 3.7, but should work fine with anything later.
